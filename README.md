# LaTeX Editor — Web Application

> Browser-based LaTeX formula editor with real-time preview and export capabilities.

## Features

- **Live Preview**: Instant rendering of LaTeX formulas
- **Design System**: Custom UI with modular styles
- **Export**: Generate publication-ready output
- **Deployment**: Ready for Vercel hosting

## Quick Start

```bash
npm install
npm run build
```

## Project Structure

```
latex/
├── index.html        # Main application
├── styles/           # CSS design system
├── scripts/          # Application logic
├── data/             # Formula templates
├── design-system/    # UI component library
├── dist/             # Build output
└── vercel.json       # Deployment config
```
