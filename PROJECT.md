# PROJECT: LaTeX Helper

## Vision
A premium, web-based productivity suite for teachers and students to effortlessly create professional LaTeX documents, MCQ exams, and TikZ geometry figures without needing deep coding knowledge.

## Goals
- Provide a "Pro Max" MCQ Wizard for exam creation.
- Implement an interactive Geometry Wizard for TikZ figure generation.
- Offer a "Table Builder Pro" for complex LaTeX tables.
- Seamless "Natural Language to LaTeX" conversion.

## Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript (ES6+).
- **Libraries**: KaTeX, ECharts (for visualizations), Google Fonts (Crimson Pro, Atkinson Hyperlegible).
- **Deployment**: Vercel.
